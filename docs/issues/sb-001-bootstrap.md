# Project Bootstrap

## Summary

Scaffold the pattern-stack Python backend inside stack-bench using `pts init` where possible, then fill in what it doesn't generate. Results in a running FastAPI app with health endpoint and passing CI.

## Scope

What's in:
- `pts init backend/` for patterns.yaml, docker-compose, .env, port isolation
- pyproject.toml with deps: pattern-stack[dev,test], agentic-patterns, fastapi, uvicorn, alembic
- config/settings.py extending BaseSettings (DATABASE_URL, ANTHROPIC_API_KEY, WEBHOOK_SECRET)
- organisms/api/app.py with create_app() factory + health endpoint
- alembic.ini + alembic/env.py (no tables yet — models come in SB-002/003/004)
- Makefile (dev, test, migrate, ci targets)
- tests/conftest.py with pattern-stack fixtures
- Empty layer directories (features/, molecules/, organisms/) with __init__.py

What's out:
- Any feature models (SB-002, SB-003, SB-004 — all parallel)
- Any routers beyond health (SB-006)
- Any business logic
- Services (added as-needed when molecules consume them)
